# Aura Gen: Nature Meditation Guide

Aura Gen is a premium, AI-powered meditation script generator designed to provide a serene and immersive breathing experience. By simply entering a location, users can generate a customized 60-second meditation session with high-end aesthetics and smooth animations.

## üöÄ Getting Started

### Prerequisites

- Node.js (v18 or higher)
- A Groq API Key

### Installation

1. Navigate to the project directory:

   ```bash
   cd nature-gen
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Configure environment variables:
   Create a `.env` file in the root directory and add your Groq API key:

   ```env
   GROQ_API_KEY=your_groq_api_key_here
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```
